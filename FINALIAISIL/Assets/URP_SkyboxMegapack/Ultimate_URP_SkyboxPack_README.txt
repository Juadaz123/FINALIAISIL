Ultimate URP Skybox Starter Pack
================================

100 ready-to-use panoramic skyboxes for Unity URP — built for simplicity and instant results.

------------------------------------------------------------
CONTENTS
------------------------------------------------------------
- 100 panoramic skybox textures (2048x1024, equirectangular)
- 100 matching Skybox/Panoramic materials
- Demo scene with skybox preview controls
- SkyboxCycler script (Left/Right arrows to cycle)

------------------------------------------------------------
INSTALLATION (UNITYPACKAGE VERSION)
------------------------------------------------------------
1. Download the `.unitypackage` file.
2. In Unity, open: Assets → Import Package → Custom Package...
3. Select the downloaded file and import all assets.
4. Everything appears under:
   Assets/URP_SkyboxStarterPack/

------------------------------------------------------------
USAGE
------------------------------------------------------------
• Open the demo scene and press ← / → to cycle skyboxes.
• To use a skybox manually:
  - Go to Window → Rendering → Lighting → Environment
  - Assign a material from Assets/URP_SkyboxStarterPack/Materials/

------------------------------------------------------------
ALTERNATIVE ZIP DISTRIBUTION
------------------------------------------------------------
If using the ZIP version, manually extract into your project.
Unity material references may need to be reconnected if .meta files are missing.

------------------------------------------------------------
TECHNICAL NOTES
------------------------------------------------------------
• Shader: Skybox/Panoramic
• Resolution: 2048×1024
• Lighting: Mipmaps enabled, GI-compatible
• Works with Unity URP 2021.3+
• No dependencies or scripts required
